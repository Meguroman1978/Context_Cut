# ⚡ Context Cut Pro - クイックスタートガイド

## 🎯 3ステップで動画クリップ生成！

Context Cut Pro を使って、たった3ステップで AI が動画を自動で切り抜き、テロップを追加できます。

---

## 📋 事前準備

### 必要なもの
- ✅ ブラウザ（Chrome, Firefox, Safari, Edge）
- ✅ 動画ファイルまたはGoogle Drive / YouTube URL
- ✅ インターネット接続

### オプション
- 📝 カスタムフォント（.ttf, .otf）
- 🎨 テロップテキスト

---

## 🚀 使い方（初回）

### Step 1: アプリにアクセス

デプロイ後のURLにアクセス:
```
https://your-app-name.streamlit.app
```

初回起動時、デフォルトフォントが自動ダウンロードされます（約5秒）。

---

### Step 2: 動画を取得

左サイドバーの「📥 動画取得」セクションで、3つの方法から選択:

#### 方法1: ローカルファイル（推奨）
1. 「ローカルファイル」を選択
2. 「動画ファイルをアップロード」をクリック
3. 動画を選択（.mp4, .mov, .avi, .mkv対応）
4. アップロード完了を待つ

#### 方法2: Google Drive
1. 「Google Drive URL」を選択
2. Google DriveのURLを貼り付け
   - 例: `https://drive.google.com/file/d/xxxxx/view`
3. 「ダウンロード」をクリック

#### 方法3: Web URL（YouTube等）
1. 「Web URL（YouTube等）」を選択
2. 動画のURLを貼り付け
   - 例: `https://www.youtube.com/watch?v=xxxxx`
3. 「ダウンロード」をクリック

> **💡 ヒント**: 初めての方は、短い動画（< 5分）でお試しください。

---

### Step 3: AI文字起こし

1. サイドバーの「🎤 AI文字起こし」セクション
2. 「文字起こしを実行」ボタンをクリック
3. 処理完了まで待つ（5分動画で約2-3分）

✅ 完了すると、「○○件のセグメントをインデックス化しました!」と表示されます。

---

## 🎬 基本的な使い方

### タブ1: 🔍 シーン検索

**自然言語でシーンを検索**

1. 検索ボックスに言葉を入力
   - 例: 「面白いシーン」「商品の説明」「感動的な場面」
2. 「検索実行」をクリック
3. 結果一覧から目的のシーンを選択
4. 「このシーンを選択」ボタンをクリック

> **✨ AI検索のコツ**: 具体的なキーワードよりも、シーンの「雰囲気」や「内容」を記述すると良い結果が得られます。

---

### タブ2: ✂️ カット範囲指定

**カット範囲を微調整**

1. 「開始時間」と「終了時間」を入力
   - または、スライダーで視覚的に調整
2. 選択範囲が表示されます（例: 15.5秒）
3. 「プレビューを生成」をクリック
4. カット結果を確認

> **💡 ヒント**: プレビューは高速に生成されるので、何度でも調整できます。

---

### タブ3: 💬 テロップ編集

**テロップを追加して完成**

#### 3-1: テキスト入力
テキストボックスにテロップを入力:
```
Context Cut Pro で
簡単動画編集！
```

#### 3-2: スタイル設定

**左カラム:**
- **フォント選択**: プルダウンから選択
  - デフォルト: Noto Sans JP
- **フォントサイズ**: スライダーで調整（24-120px）
- **文字色**: カラーピッカーで選択

**右カラム:**
- **背景**: 
  - なし（透明）
  - 黒（半透明）← 見やすくておすすめ
  - 白
- **テロップ位置**:
  - 下部中央 ← 標準
  - 上部中央
  - 中央

#### 3-3: カスタムフォントを追加（オプション）
1. 「新しいフォントを追加」セクション
2. フォントファイル（.ttf, .otf）をアップロード
3. 「フォントを追加」をクリック
4. プルダウンに即座に追加されます

#### 3-4: 動画生成
1. 「🎬 テロップ付き動画を生成」ボタンをクリック
2. 処理完了まで待つ（約30-60秒）
3. プレビューが表示されます
4. 「📥 動画をダウンロード」でエクスポート

🎉 **完成！** MP4ファイルがダウンロードされます。

## 📱 実践例

### 例1: YouTubeハイライト作成

**目的**: 長時間YouTube動画から面白いシーンを切り抜く

1. YouTube URLを入力
2. 文字起こし実行
3. 検索: 「面白い」「笑い」「爆笑」
4. 結果から最適なシーンを選択
5. カット範囲を微調整
6. テロップ: 「爆笑シーン集！」
7. 生成してSNSにシェア

**所要時間**: 約5-10分

---

### 例2: セミナー動画の重要ポイント抽出

**目的**: 1時間のセミナーから3つのキーポイントを抽出

1. セミナー動画をアップロード
2. 文字起こし実行
3. 検索1: 「重要なポイント」
4. シーン1を選択 → テロップ追加 → 生成
5. 検索2: 「まとめ」
6. シーン2を選択 → テロップ追加 → 生成
7. 3つのクリップを作成

**所要時間**: 各クリップ約5分 × 3 = 15分

---

### 例3: 商品紹介動画の作成

**目的**: 製品デモ動画から特徴を強調したクリップ作成

1. 製品デモ動画をアップロード
2. 文字起こし実行
3. 検索: 「新機能」「特徴」「メリット」
4. 各シーンを選択してカット
5. テロップでポイントを強調:
   - 「驚きの新機能！」
   - 「これだけで解決！」
6. ブランドカラーで統一
7. 生成して広告に使用

**所要時間**: 約10-15分

## ⚡ ショートカットとTips

### 効率的な検索
```
✅ Good: 「感動的な場面」「商品の説明部分」
❌ Bad: 「いい」「あれ」
```

### カット範囲の調整
- **数値入力**: 秒単位で正確に指定
- **スライダー**: 視覚的に大まかな範囲を調整

### テロップのベストプラクティス
- **文字数**: 1-2行（約20文字以内）が見やすい
- **背景**: 黒（半透明）が最も視認性が高い
- **位置**: 下部中央が標準

### パフォーマンス最適化
- **動画サイズ**: < 500MB推奨
- **動画長さ**: < 10分推奨
- **処理時間**: 5分動画 ≈ 5分処理

## 🐛 よくある質問（FAQ）

### Q1: 文字起こしに時間がかかります
**A**: Whisper AIはCPU処理のため、動画の長さに比例して時間がかかります。
- 5分動画: 約2-3分
- 10分動画: 約5-10分
- 30分動画: 約15-30分

### Q2: 検索結果が表示されません
**A**: 以下を確認してください:
- 文字起こしが完了しているか
- 検索クエリが具体的か（「面白い」より「笑っている」の方が良い）
- 動画に音声が含まれているか

### Q3: プレビューが生成されません
**A**: 
- カット範囲が0秒ではないか確認
- 開始時間 < 終了時間になっているか確認
- ブラウザを更新してみる

### Q4: テロップが文字化けします
**A**:
- 日本語対応フォント（Noto Sans JP等）を選択
- カスタムフォントは日本語対応のものを使用

### Q5: ダウンロードした動画が再生できません
**A**:
- MP4形式に対応したプレイヤーを使用
- VLC Media Player（無料）の使用を推奨
