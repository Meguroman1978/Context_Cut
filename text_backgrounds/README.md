## カスタム背景の追加方法

### 推奨素材サイト

1. **マンガパーツSTOCK** (https://mangasozai.com/)
   - 漫画的な吹き出しや背景素材
   - 無料・商用利用可能
   - PNG形式で透過背景対応

2. **マンガ文字素材dddFont** (https://dddfont.com/)
   - 擬音（オノマトペ）素材
   - 無料・商用利用可能
   - 文字背景としても使用可能

### 追加手順

1. 素材サイトから背景画像をダウンロード
2. PNG または JPG 形式で保存
3. このディレクトリ (`text_backgrounds/`) に配置
4. アプリを再起動すると自動的にプリセットに表示されます

### 画像の推奨仕様

- **形式**: PNG（透過対応）または JPG
- **サイズ**: 400x150px 程度（自動スケーリング対応）
- **ファイル名**: 半角英数字とアンダースコア（例: `my_background.png`）
- **透明度**: PNG形式でアルファチャンネルを使用可能
